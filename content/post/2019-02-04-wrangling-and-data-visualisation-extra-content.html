This dataset contains game-by-game statistics of players from 2000 to 2017.

Many of the #TidyTuesday posts I saw on twitter when filtered between the dates where this was the active dataset made the conclusion that modern NFL has a trend towards greater emphasis on the passing game (plays where the Quarterback throws the ball) compared to the running game (when the QB hands the ball off to a Running Back).

To build on these suggestions and build a similar plot to the golden-age TV plot we saw above let‚Äôs produce a visualisation showing the total yards gained by every RB for each sesion we have data for. To work out the top 10 running seasons in this 17-year span we must first summarise each player‚Äôs performance per season in terms of total yards gained (also total attempts as additional information). The commands here are mostly familiar from the lecture/workshop material we have already covered. The things not seen already are grouping by multiple variables at once and the ungroup() function. This group_by command will segment the dataframe by unique combinations of name and year meaning our summary operations will be calculated on each player for each season, without game_year we would be calculating the total number of yards acquired over all the seasons for which the player has data. After the data is summarised it‚Äôs a useful habit to ungroup() the dataframe if you are saving it to the workspace just incase you forget it‚Äôs grouped and it creates unintended consequences down the line.

top_rushers <- nfl.data %>% 
  group_by(name, game_year) %>% 
  summarise(totalRushYds = sum(rush_yds, na.rm=TRUE),
            totalRushAttempts = sum(rush_att, na.rm=TRUE)) %>% 
  arrange(desc(totalRushYds)) %>% 
  ungroup() %>% 
  slice(1:10)

Now we want to create a dataframe to use for the main plot that filters out any player with less than 50 carries and we create a new variable that is used to flag whether or not a particular player is one of the players who tallied one of the 10 best rushing seasons.

summarised.runner.data <- nfl.data %>% 
  group_by(name, game_year) %>% 
  summarise(totalRushYds = sum(rush_yds, na.rm=T),
            totalRushAttempts = sum(rush_att, na.rm=T)) %>% 
  filter(totalRushAttempts > 50) %>%
  ungroup() %>% 
  mutate(top_runner = ifelse(name %in% unique(top_rushers$name), "highlight", "noHighlight"))

Now that we know the runners who had the best 10 seasons in the 2000-2017 span and this is flagged in a summary dataset we can create the plot. Similar to the workshop we‚Äôll do this in several stages looking at the results as we go.

Step One - Basic Plot Structure

Create the basic structure of the ggplot. for each game_year we plot the rushing totals for all players with over 50 total attempts. As game_year is discrete we add position=position_jitter() as this adds a slight bit of randomness to the location where the dot is plotted - we do this because of the amount of overlapping points. We have seen geom_smooth() already but with method=‚Äúlm‚Äù for a linear fit. Here we use method=‚Äúloess‚Äù which is a localised regression fit meaning it will follow the trend of the data unconstrained. The annotate function adds the ‚Äòoverall trend‚Äô label to the plot at the x and y coordinates specified.

main.colour = "#43a3ca" # hex color used repeatedly so define as variable

best_season_plot <- summarised.runner.data %>% 
  ggplot(aes(x = game_year, y = totalRushYds, size = totalRushAttempts)) +
  geom_point(aes(alpha = as.factor(top_runner)), color = main.colour, position = position_jitter(width=0.1)) + 
  geom_smooth(method="loess", se=F, color="grey4", linetype="dashed", size=1, alpha=0.7) + 
  annotate("text",
           x = 2014, y = 500, color = "grey4",
           label = "Overall Trend", size = 8, fontface ="bold",
           family="Calibri",
           alpha = 0.6
  )
best_season_plot
## Warning: Using alpha for a discrete variable is not advised.

Step Two

Use geom_path() to link the top runners data together. We include a data argument inside the geom because we want this geom to use a subset of the main dataset. By only including the data of those who are top_runners we ensure that this geom_path() layer will only be drawn for the ‚Äútop runners‚Äù.

best_season_plot <- best_season_plot + 
  geom_path(
    data = filter(summarised.runner.data, top_runner == "highlight"),
    aes(group = name), color = main.colour, size = 0.75
  )
best_season_plot
## Warning: Using alpha for a discrete variable is not advised.

Step Three

Add player name labels for those runners whom we have highlighted. Note here the data selection - we want the player label to point to the runners best season.

best_season_plot <- best_season_plot + 
  geom_text_repel(
    data = filter(summarised.runner.data, top_runner == "highlight") %>% 
      group_by(name) %>% 
      arrange(desc(totalRushYds)) %>% 
      slice(1),
    aes(label = name),
    size = 6,
    nudge_y = + 300,
    nudge_x = 0.5,
    segment.size = 0.5,
    segment.colour = "black",

Step Four

We have all the information we need but the appearance is a little bit all over the place. So we need to tidy that up. A lot of this is new as we spent the workshop time gaining experience of creating standard ggplots needing minimal to no customisation.

The scale_x_ and scale_y_ family of functions are used to alter the axes of the plots.

In the initial setup of the ggplot we set the alpha mapping to be as.factor(top_runner) because we want the points that aren‚Äôt represnting the top runners to be much more transparent than the top runners. The default vlaues selected weren‚Äôt suitable because the top runners data was the category selected to be more transparent (we could have releveled the factor but better to learn this way imo). We use scale_alpha_discrete to manually set the alpha values for each category. The *guide=‚Äúnone‚Äù removes the legend for this mapping.

best_season_plot <- best_season_plot  +
  scale_x_continuous(breaks = seq(2000,2017,2)) + 
  scale_y_continuous(position = "right", limits=c(0,2400)) + 
  scale_alpha_discrete(range=c(0.9,0.2), guide = "none") + 
  labs(x = "Season",
       y = "Total Rush Yards",
       title = "Every rushing yardage total 2000 - 2017 (min 50 attempts)",
       subtitle = "Emphasis on rushers with the top 15 best rushing seasons in that timespan",
       caption = "Data: #TidyTuesday 2018 week 22")
## Warning: Using alpha for a discrete variable is not advised.
best_season_plot

Step Five

All purely cosmetic changes to tweak the final appearance. the use of rel() to define the text sizes is new for us and useful. It means to change the size relative to the overall default size, so think of rel(1) as being the default text size and then rel(0.%) would be half that size whereas rel(2) would be twice that size.

best_season_plot <- best_season_plot + 
  theme_minimal() + 
  theme(
    text = element_text(family = "Calibri"),
    legend.position = c(0.5,0.05),
    legend.direction = "horizontal",
    axis.text = element_text(size = rel(1)),
    title = element_text(size = rel(2), face="bold"),
    legend.text = element_text(size=rel(.9)),
    legend.title = element_text(size=rel(.75)),
    axis.title = element_text(size = rel(1.5))
  )
best_season_plot

As we chose to print the name labels above the best season a player had it means they are fairly crowded so this plot could be better with more customisation with regards to the label position. Nevertheless it‚Äôs a nice visualisation showing the overall trend of rushing totals while highlighting standout performers.
